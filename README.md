# ⚡ DC-DC Boost Converter with PWM Control (12V → 24V)

## 📌 Overview
This project presents the design and implementation of a DC-DC boost converter that steps up a 12V input to 24V output. The system is controlled using PWM generated from a microcontroller (ATtiny85), with gate driving through an opto-isolated driver.

The design is validated through theoretical calculations, simulation, and hardware implementation.

## 🎯 Design Specifications

| Parameter | Value |
|----------|------|
| Input Voltage (Vin) | 12V |
| Output Voltage (Vout) | 24V |
| Switching Frequency | 10 kHz |
| Load Resistance | 32 Ω |
| Duty Cycle | 0.5 |
| Inductor | 1.25 mH |
| Capacitor | 220 µF |

## ⚙️ Working Principle

A boost converter increases voltage using switching:

### 🔹 Switch ON
- Inductor stores energy  
- Diode is OFF  
- Load powered by capacitor  

### 🔹 Switch OFF
- Inductor releases energy  
- Output voltage increases  
- Diode conducts

## 🛠️ Hardware Implementation

The circuit includes:

- MOSFET (IRF540N) as switching device  
- Gate driver (TLP250) for isolation  
- Inductor for energy storage  
- Fast diode (MUR460) for switching  
- Output capacitor for ripple reduction  
- ATtiny85 for PWM generation
